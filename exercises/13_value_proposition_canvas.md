# 13. Value Proposition Canvas

## Purpose

Summarize stakeholder jobs, pains, and gains so later ideation can respond to
real needs without prematurely defining a solution.

## Inputs

- Stakeholder map
- Audience segments
- Interview, photojournal, power, and resource-flow claims
- Active challenge frame and impact goals

## Tasks

1. Identify each stakeholder role that needs a persona-style map.
2. Extract jobs to be done, pains, gains, constraints, and motivations.
3. Separate validated human evidence from agentic inference.
4. Identify where stakeholder needs diverge or conflict.
5. Prepare value hypotheses as inputs for later ideation, not as final
   solutions.

## Expected Outcome

- Persona mapping for each relevant stakeholder.
- Jobs to be done.
- Pains and gains.
- Divergent needs across roles.
- Value hypotheses and validation gaps.

## Provenance Requirements

Each persona element must cite supporting claims. Deprecated or contradicted
claims cannot be used unless explicitly superseded by stronger later evidence.
